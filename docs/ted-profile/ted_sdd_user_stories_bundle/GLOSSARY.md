- **Draft-only**: Ted may create drafts/proposals, but never sends email or invites.
- **Approval / certification**: Explicit operator action required before risky external writes (e.g., applying filing or creating holds).
- **Deal ledger**: Canonical list of deals used to link emails/docs/tasks to a deal context.
- **Triage / Unfiled Work**: Queue for outputs that lack required linkage to a deal or task.
- **Profile**: A business identity context (e.g., Olumie vs Everest) used for Graph and file contexts.
- **Job family**: Governance category that sets default tool posture and allowed mode ladder (GOV, ING, LED, OUT, MNT).
