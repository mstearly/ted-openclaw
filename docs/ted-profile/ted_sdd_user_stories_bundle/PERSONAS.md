Primary and only end-user for day-1 operation. Needs a chat-first assistant that drafts (never sends), organizes deal work, and surfaces deadlines.

Supports tenant app registration/consent, least-privilege scopes, audit logging expectations, and rapid revocation.

## Support Engineer / Operator Coach

Helps with packaging, install, troubleshooting (Doctor + logs), and release validation gates.
